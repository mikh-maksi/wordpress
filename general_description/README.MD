# Програма підготовка зі створення сайтів під WordPress.
Ключова задача освітньої програми в тому, щоб людина, що володіє навичками Web-розробки зміг почати працювати зі створенням сайтів під WordPress.
Для підготовки сформовано flow робіт від найпростіших до найскладніших по роботі із сайтом:
* вхід
* адміністрування інформації
* Пошук, встановлення та наповлення інформації відповідних тем
* корегування оформлення: CSS, JS
* Пошук, встановлення та адптація потрібних плагінів
* Створення та встановлення власних тем
* Створення  та встановлення власних плагінів.
